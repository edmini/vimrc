## 키보드 단축키

ctrl + h, j, k, l (왼쪽, 아래, 위, 오른쪽)

### 버퍼이동 - vim tab 이동
ctrl + u, i, w (왼쪽 탭, 오른쪽 탭, 탭 닫기)

### NERDTree
