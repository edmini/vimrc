## 키보드 단축키
### 창이동
ctrl + h, j, k, l (왼쪽, 아래, 위, 오른쪽)

### 버퍼이동 - vim tab 이동
ctrl + u, i, w (왼쪽 탭, 오른쪽 탭, 탭 닫기)

### NERDTree
ctrl + t (NerdTree 토글)

### Telescope
#### leader 설정 : 스페이스바 키
leader + ff

### vim term
#### leader 설정 : 스페이스바 키
leader + tt

### autoformat
#### leader 설정 : 스페이스바 키
leader + aa

### tagbar
F8
